## Top Languages
![Top Languages](https://github-readme-stats.vercel.app/api/top-langs/?username=SahilBacchus&layout=compact&theme=tokyonight)


> Notes:  
> - username=SahilBacchus -> your GitHub username  
> - layout=compact -> displays the language stats in a compact view  
> - theme=tokyonight -> applies the Tokyonight color theme  

> more params:
> - hide=HTML,CSS -> hide specific languages  
> - langs_count=5 -> number of top languages to show

> repo: https://github.com/anuraghazra/github-readme-stats
